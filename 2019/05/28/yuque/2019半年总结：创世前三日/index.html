2019半年总结：创世前三日
写于2019年05月28日

上帝花了7天创造世间万物。
第一天「要有光！」于是便有了白天和黑夜。
第二天将地表的水和空气中的水分隔开，这便有了天空。
第三天把地表的水汇集成了海洋，裸露的便是陆地，并在陆地上长出植物。

他工作到一半已经孕育了生命体，也为动物生存环境奠定了基础。
我2019过去的半年也创造了新生活的基础：从亚洲跑到欧洲生活。

2018年总结仿佛还在昨天，2019已经过半了：

  1. 台北跨年/Vlog初体验
  2. 入职蚂蚁金服
  3. 欧洲9日初体验
  4. 新疆团建自驾游
  5. 离职搬到荷兰
  6. 读书20本，电影35部

想做什么就去做

年初做了几个Vlog很花时间，卡一个音乐节点会重复个几十遍。但也很有成就感。
做完之后分享环节遇到B站账户注册不了就放弃了。这两天又重新注册跑来跑去的阿乐，把之前的视频翻出来上传很糊但完全没有之前的热情去搞了。所以还是要抓住那一点微弱的激情。

坚持

背单词断了，唯一还保留着的习惯是起床后喝杯温开水和记账。
为什么坚持不了呢？明明任务很轻松每天只要花几分钟就能完成。
重新看《意志力》这本书提到的科学方法，审视自己还是因为缺少明确的目标。我是漫无目的无所事事的懒人性格。做什么就是体验过就好，没有目标没有斗志。对于人生的终极目标不清晰，或者对背单词、做瑜伽的目标不清晰，这就导致我非常容易放弃。
是时候伪造一个目标贴在床头时刻提醒自己了！

思考

在新团队待的时间很短暂但获益匪浅，一不小心看了20本书也是意料之外。
列下可以称之为输出的东西（少得可怜）：

  • 假期里的郁金香
  • 富兰克林自传
  • 流浪地球

最重要的收获是了解到优秀的人也是从普通人一点一点努力得到的。
不是他们有什么过人之处，而是每个细节都比你努力一点点。
要有自己独立思考能力，并刻意练习。只要你足够地想得到那样的人生，你就可以收获那样的人生。

为什么不开公众号？

不以此谋生，不因此讨好别人。
做自己喜欢的事情，做让自己变得更好的事情。
文字是开放的，坚持做个开放的博主。

  • 多写书评，多写影评（2018总结也是这么说的
  • 体验欧洲生活
